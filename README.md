# 목표

글을 공유하는 게시판의 성격을 갖지만, 더 심플하고 직관적인 UI/UX를 지향합니다.

게시글 비공개 기능을 활용하여 글을 투고할 때 작업 공간으로 활용할 수 있도록 합니다.

# 환경변수

> [!NOTE]
> 빠른 서버측 데이터 페칭을 위해 백엔드 호스트를 가르키는 환경변수를 분리합니다.
> `INTERNAL_API_URL`은 백엔드 서버가 동일 네트워크 안에 있다면 내부망 주소를 사용하세요.

```
NEXT_PUBLIC_API_URL=
INTERNAL_API_URL=
NEXT_PUBLIC_RECAPTCHA_SITE_KEY=
```
